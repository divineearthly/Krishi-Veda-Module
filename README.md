# Krishi-Veda AI Platform

## Architecture
- **C++ Kernels**: High-performance Vedic Sutra implementations for edge devices.
- **Python Bridge**: ctypes-based interface for AI-Vedic hybrid computing.
- **Ahimsa Engine**: Sustainability protocol for organic farming.

## Deployment
1. Compile kernels: `g++ -O3 -shared -fPIC -o libvedic.so vedic_kernels.cpp`
2. Install dependencies: `pip install scikit-learn pandas numpy`
3. Run: `python main.py` for farmer recommendations.
